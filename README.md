# eBay Kleinanzeige Scraper

Приложение для сбора данных с eBay Kleinanzeige с интерфейсом.

## Возможности

- Поиск объявлений по различным категориям
- Фильтрация по местоположению
- Извлечение информации: название, цена, описание, местоположение, дата публикации, URL изображений
- Сохранение данных в форматах JSON и CSV
- Красивый и интуитивно понятный интерфейс с поддержкой темной и светлой темы
- Настройка задержек между запросами
- Возможность использования прокси

## Установка

1. Клонируйте репозиторий:
git clone https://github.com/yourusername/ebay-kleinanzeige-scraper.git
cd ebay-kleinanzeige-scraper


2. Установите зависимости:

pip install -r requirements.txt


## Запуск

### Графический интерфейс

Для запуска приложения с графическим интерфейсом:

python ebay_kleinanzeige_gui.py


### Консольный режим

Для запуска в консольном режиме (без интерфейса):

python ebay_kleinanzeige_scraper.py


## Использование

1. Выберите категорию для поиска
2. Укажите местоположение (опционально)
3. Настройте количество страниц и задержки
4. Нажмите "Начать сбор данных"
5. Результаты будут отображены в таблице и могут быть экспортированы в JSON или CSV

## Этический скрапинг

Этот инструмент предназначен для образовательных целей и личного использования. Пожалуйста, придерживайтесь следующих принципов:

- Устанавливайте разумные задержки между запросами (рекомендуется минимум 3-5 секунд)
- Ограничивайте количество запросов в день
- Соблюдайте правила использования сайта (robots.txt)
- Используйте данные только для личных, некоммерческих целей
